How to Play

Fill the 9×9 grid with numbers so that each row, column, and 3×3 box contains all digits from 1 to 9 without repetition.

  1. Step 1: Click on an empty cell to select it
  2. Step 2: Enter a number using the number pad or keyboard
  3. Step 3: Red numbers indicate conflicts
  4. Step 4: Use "Check" to verify progress
  5. Step 5: Use "Hint" if stuck (3 max)
